Forge Technologies Project Setup

INITIAL CONFIG

Prerequisites

ensure you have the following installed:

Node.js

https://nodejs.org/en/download

Ruby

http://rubyinstaller.org

Bower

npm install -g bower

!!!!! NB !!!!!

Before you carry on, set up your github project in Sourcetree or Github Desktop

For Forge purposes, we will set the local repo path to Projects/Project Name/3_root

for Windows

run CMD as administrator, and run the following commands:

npm install

bower install

npm install gulp -g

npm install gulp

gulp install-js

gulp concat-js

that should complete the setup, next is to run gulp

gulp

for Mac

run Terminal, and run the following commands:

./update.sh

INSTALLING JS COMPONENTS

run CMD as administrator, and run the following command:

bower install --save-dev MODULENAME

This installs the module to your "_system/_bower.cache" folder

in your dev client open _config.json

in the bower section of the code, add the path to the .js file ...

    
        {
            "root": "www",
                "assets" : {
                    "js" : {
                        "bower" : [
                            "path/filename.js",
                            "path/filename.js"

and futher down in the order section of the code, add .js file

    
                        "order" : [
                            "filename.js"
                            "filename.js"
                            "filename.js"

!!!!! NB !!!!!

important to note that this section sets the precedence of the JS files.

Therefore if the .JS file you are loading is required by other components, it should be loaded ABOVE those which require it

save that, and head back over to CMD/Terminal, and run the following commands:

gulp install-js

which copies all the JS files to our assets/js/components directory

gulp concat-js

which compiles the assets/js/components.js file.



Thats it ... go be AWESOME!!!
